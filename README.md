# Superstore Sales Analysis Using SQL

## Objective
Analyze sales data using SQL with filtering, aggregation, and business queries.

## Tasks Performed
- Loaded Superstore dataset into SQLite database
- Explored schema and sample data
- Applied WHERE filters on Region, Category, Date, and Sales
- Used GROUP BY for aggregations
- Identified top products and categories
- Analyzed monthly sales trends
- Found top customers
- Detected duplicate orders
- Performed data quality validation

## Technologies Used
- Python
- Pandas
- SQLite
- SQL
- Kaggle Notebook

## Dataset
Superstore Dataset (Kaggle)

## Key Insights
- Technology category generated high sales.
- Sales varied significantly across regions.
- A small number of customers contributed a large portion of revenue.
- Monthly sales showed changing trends over time.
- Data quality checks indicated minimal missing values.
